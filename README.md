## 🛠 Installation
To install Goat-Bot-V3, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Lord9s/Bestbot.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Bestbot
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Run the bot:
   ```bash
   npm start
   ```
